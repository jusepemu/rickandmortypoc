About this

This project is a technological adventure inspired by the Rick and Morty universe, where the goal is to go beyond the conventional and challenge the limits of web development with Angular.

Why did you build this?

The main challenge of this POC is to explore the combination of Server-Side Rendering (SSR) and Single-Page Application (SPA) in Angular, facing the challenge of achieving a smooth and fast experience both on the server and the client. This approach aims to demonstrate how Angular can adapt to different needs, optimizing performance, accessibility, and user experience, while experimenting with the integration of ZardUI and a simple, functional folder structure.

The decision to address the hybrid SSR/SPA mode responds to the need to offer applications that load quickly, are SEO friendly, and maintain a dynamic experience. This POC is an opportunity to learn, make mistakes, and improve, laying the foundation for more ambitious and robust future projects.

Throughout the development, best practices, clarity in code organization, and ease of maintenance are prioritized, without losing sight of the importance of experimenting and pushing the framework's boundaries.

Main features

  • Home page as a starting point.
  • Character list with smart filters.
  • Enriched detail for each character.
  • Favorites management to personalize the experience.

If you are curious about the architecture, decision-making, or lessons learned from this POC, feel free to explore the code or get in touch to share ideas and suggestions.
