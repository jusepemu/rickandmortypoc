The Challenge

The challenge of this project is to create a proof of concept based on the Rick and Morty API, using Angular as the main framework. The goal is for each participant to explore and compare different architectures and development methodologies, testing both performance and implementation speed, as well as code organization. This experience will help identify the advantages and challenges of each approach, focusing not on visual design, but on the structure and functionality of the application.
